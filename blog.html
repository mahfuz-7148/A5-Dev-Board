Question-1: What are the different ways to select an element in the DOM?

document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector(), document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML sets or gets HTML content, innerText handles visible text only, and textContent grabs all text, even hidden, without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events triggered by its child elements, leveraging event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process in the DOM where an event triggered on an element propagates upward through its parent elements in the hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: const newElement = document.createElement('div'); Add: document.body.appendChild(newElement); Remove: newElement.parentNode.removeChild(newElement);
